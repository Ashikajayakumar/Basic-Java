<h2 align="center"><b>1. Variable Declaration</b></h2>

A variable in Java is a <b>container</b> that holds data of a specific type.

Syntax:

```dataType variableName;```


Example:
```
int age;       // declares a variable 'age' of type int
double salary; // declares a variable 'salary' of type double
char grade;    // declares a variable 'grade' of type char
```

Here,` int`,` double`, `char` are data types, and `age`, `salary`, `grade` are variable names.

<h2 align="center"><b> 2. Variable Initialization </b></h2>

Initialization means <b> assigning a value to a variable.</b> This can be done at the time of declaration or later.

Syntax:
```
variableName = value;
```

Example:
```
age = 25;          // assigns 25 to age
salary = 45000.50; // assigns 45000.50 to salary
grade = 'A';       // assigns 'A' to grade
```

You can also <b> declare and initialize</b>  together:
```
int age = 25;
double salary = 45000.50;
char grade = 'A';
```
<h2 align="center"><b> 3. Variable Naming Rules in Java </b></h2>

* When naming a variable in Java, follow these rules:
  
<h5 align="center"><b> 1.Must start with a letter,` $`, or` _`</b> 
  
```
int age;       // valid
int _count;    // valid
int $salary;   // valid
```

Invalid:

```
int 2age;      // invalid: starts with a number
```

<h5 align="center"><b> 2. Can contain letters, `digits`,` $`, or` _` </b></h5>

```
int myAge2;     // valid
int total$;     // valid
int _value_1;   // valid
```

<h5 align="center"> <b> 3. Cannot be a Java keyword </b></h5>

Keywords are reserved words like ``int``, ``class``,``if``,`` for``, etc.
```
int class;  // invalid
```

<h5 align="center"> <b> 4. Case sensitive </b></h5>

int age;  
int Age;   // different variable


<h5 align="center"> <b> 5.Should be meaningful (good practice) </b></h5>
```
int score;   // good
int x;       // not recommended
```

                                               Literal


* A literal have a constant value  fixed data type and often known as constant in Java.
* Literal assigned to a variable to provide a value to variable.


<h2 align="center"><b>A. Integer Literals</b></h2>

Integer literals are numbers without decimals. They can be written in 3 formats:

<h5 align="center"> <b>1.Decimal (base 10) – normal numbers</b></h5>
```
int dec = 100;  // decimal literal
```

<h5 align="center"> <b>2.Octal (base 8) – starts with` 0 `</b></h5>

```
int oct = 010;  // octal literal → 010 = 8 in decimal
```

<h5 align="center"> <b>3.Hexadecimal (base 16) – starts with ` 0x or 0X`</b></h5>

```
int hex = 0x1A; // hexadecimal literal → 1A = 26 in decimal
```

Note: <b> Binary literals (base 2)</b> are also supported in Java 7+ using `0b or 0B`:
```
int bin = 0b1010; // binary literal → 10 in decimal
```
<h2 align="center"><b>B. Floating-Point Literals</b></h2>
 

Used for<b> numbers with a fractional part</b> (decimal numbers).
```
double price = 99.99;      // default is double
float weight = 65.5f;      // float literal must use 'f' or 'F'
```
* Default floating-point is double.
* Use <b>f/F</b> for float literals.
* Use <b>d/D</b> optionally for double literals.

<h2 align="center"><b> C. Character Literals </b></h2>

* <b>A single character enclosed in single quotes `' '`.</b>
```
char grade = 'A';
char symbol = '@';
char digit = '5';
```

* Can also use <b>escape sequences</b>:
```
char newline = '\n';
char tab = '\t';
char backslash = '\\';
```
<h2 align="center"><b> D. String Literals </b></h2>

* <b>A sequence of characters enclosed in double quotes `" "`.</b>
```
String name = "Ashika";
String greeting = "Hello, World!";
String empty = ""; // empty string literal
```


* Strings are objects in Java `(java.lang.String)`.

<h2 align="center"><b> E. Boolean Literals </b></h2>

Boolean literals represent <b>true/false </b> values.
```
boolean isJavaFun = true;
boolean isRaining = false;
```

Only `true` or `false` are valid literals for boolean type.
