<h2 align="center"><b>1. Variable Declaration</b></h2>

A variable in Java is a <b>container</b> that holds data of a specific type.

Syntax:

```dataType variableName;```


Example:
```
int age;       // declares a variable 'age' of type int
double salary; // declares a variable 'salary' of type double
char grade;    // declares a variable 'grade' of type char
```

Here,` int`,` double`, `char` are data types, and `age`, `salary`, `grade` are variable names.

<h2 align="center"><b> 2. Variable Initialization </b></h2>

Initialization means <b> assigning a value to a variable.</b> This can be done at the time of declaration or later.

Syntax:
```
variableName = value;
```

Example:
```
age = 25;          // assigns 25 to age
salary = 45000.50; // assigns 45000.50 to salary
grade = 'A';       // assigns 'A' to grade
```

You can also <b> declare and initialize</b>  together:
```
int age = 25;
double salary = 45000.50;
char grade = 'A';
```
<h2 align="center"><b> 3. Variable Naming Rules in Java </b></h2>

* When naming a variable in Java, follow these rules:
  
<b> 1.Must start with a letter,` $`, or` _`</b> 
  
```
int age;       // valid
int _count;    // valid
int $salary;   // valid
```

Invalid:

```
int 2age;      // invalid: starts with a number
```

<b> 2. Can contain letters, `digits`,` $`, or` _` </b>
```
int myAge2;     // valid
int total$;     // valid
int _value_1;   // valid
```

<b> 3. Cannot be a Java keyword </b>
Keywords are reserved words like ``int``, ``class``,``if``,`` for``, etc.
```
int class;  // invalid
```

<b> 4. Case sensitive </b>

int age;  
int Age;   // different variable


<b> 5.Should be meaningful (good practice) </b>
```
int score;   // good
int x;       // not recommended
```
