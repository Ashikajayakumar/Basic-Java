# Arithmetic Operators in Java

Used to perform **mathematical operations**.

| Operator | Description         | Example    |
| -------- | ------------------- | ---------- |
| `+`      | Addition            | 5 + 3 = 8  |
| `-`      | Subtraction         | 5 - 3 = 2  |
| `*`      | Multiplication      | 5 * 3 = 15 |
| `/`      | Division            | 10 / 2 = 5 |
| `%`      | Modulus (remainder) | 10 %       |
