# Java Operators Guide

This document provides a clear overview of all major Java operators with examples.

---

## 1. Arithmetic Operators

Used to perform **mathematical operations**.

| Operator | Description         | Example      |
| -------- | ------------------- | ------------ |
| `+`      | Addition            | `5 + 3 = 8`  |
| `-`      | Subtraction         | `5 - 3 = 2`  |
| `*`      | Multiplication      | `5 * 3 = 15` |
| `/`      | Division            | `10 / 2 = 5` |
| `%`      | Modulus (remainder) | `10 % 3 = 1` |

**Example:**

```java
int a = 10, b = 3;
System.out.println(a + b); // 13
System.out.println(a % b); // 1
```

---

## 2. Unary Operators

Used to **increment, decrement, or negate** values.

| Operator | Description    | Example         |
| -------- | -------------- | --------------- |
| `++`     | Increment by 1 | `a++` or `++a`  |
| `--`     | Decrement by 1 | `a--` or `--a`  |
| `!`      | Logical NOT    | `!true = false` |

**Pre vs Post Increment/Decrement**

```java
int x = 5;
System.out.println(++x); // pre-increment, output: 6
System.out.println(x++); // post-increment, output: 6, x becomes 7
```

---

## 3. Relational Operators

Used to **compare values**.

| Operator | Description      | Example         |
| -------- | ---------------- | --------------- |
| `==`     | Equal to         | `5 == 5` → true |
| `!=`     | Not equal to     | `5 != 3` → true |
| `>`      | Greater than     | `5 > 3` → true  |
| `<`      | Less than        | `5 < 3` → false |
| `>=`     | Greater or equal | `5 >= 5` → true |
| `<=`     | Less or equal    | `3 <= 5` → true |

---

## 4. Logical Operators

Used for **boolean logic**.

| Operator | Description | Example                 |            |       |   |               |
| -------- | ----------- | ----------------------- | ---------- | ----- | - | ------------- |
| `&&`     | Logical AND | `true && false = false` |            |       |   |               |
| `        |             | `                       | Logical OR | `true |   | false = true` |
| `!`      | Logical NOT | `!true = false`         |            |       |   |               |

> **Short-circuit behavior:**

* `&&` stops evaluation if **first operand is false**
* `||` stops evaluation if **first operand is true**

---

## 5. Assignment Operators

Used to **assign values** to variables.

| Operator | Meaning              | Example                |    |      |
| -------- | -------------------- | ---------------------- | -- | ---- |
| `=`      | Assign               | `a = 5`                |    |      |
| `+=`     | Add & assign         | `a += 3` → `a = a + 3` |    |      |
| `-=`     | Subtract & assign    | `a -= 2`               |    |      |
| `*=`     | Multiply & assign    | `a *= 2`               |    |      |
| `/=`     | Divide & assign      | `a /= 2`               |    |      |
| `%=`     | Modulus & assign     | `a %= 3`               |    |      |
| `&=`     | Bitwise AND & assign | `a &= 2`               |    |      |
| `        | =`                   | Bitwise OR & assign    | `a | = 3` |
| `^=`     | Bitwise XOR & assign | `a ^= 1`               |    |      |
| `<<=`    | Left shift & assign  | `a <<= 1`              |    |      |
| `>>=`    | Right shift & assign | `a >>= 1`              |    |      |

---

## 6. Bitwise Operators (Basic Understanding)

Operate on **binary representation of integers**.

| Operator | Description          | Example                 |    |        |
| -------- | -------------------- | ----------------------- | -- | ------ |
| `&`      | AND                  | `5 & 3 = 1`             |    |        |
| `        | `                    | OR                      | `5 | 3 = 7` |
| `^`      | XOR                  | `5 ^ 3 = 6`             |    |        |
| `~`      | NOT                  | `~5 = -6`               |    |        |
| `<<`     | Left shift           | `5 << 1 = 10`           |    |        |
| `>>`     | Right shift          | `5 >> 1 = 2`            |    |        |
| `>>>`    | Unsigned right shift | `-5 >>> 1 = 2147483645` |    |        |

---

## 7. Ternary Operator

A **short form of if-else**.

**Syntax:**

```java
condition ? valueIfTrue : valueIfFalse;
```

**Example:**

```java
int a = 10, b = 20;
int max = (a > b) ? a : b;
System.out.println(max); // 20
```

---

## 8. String Concatenation using `+`

The `+` operator **joins strings**.

```java
String firstName = "Ashika";
String lastName = "J";
String fullName = firstName + " " + lastName;
System.out.println(fullName); // Ashika J

int age = 25;
System.out.println("Age: " + age); // Age: 25
```
