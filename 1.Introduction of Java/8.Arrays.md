<h2 align="center"><b>  📦 7. Arrays in Java </b></h2>

<h3 align="center"><b> 🔹 What is an Array? </b></h3>

* An array is a collection of homogeneous (same type) elements stored in contiguous memory locations.
* It allows storing multiple values under a single variable name, accessed using index.`

 <h4 align="center"><b> 🔹 Key Features of Arrays: </b></h4>

* Fixed Size: Size must be known at compile time. 
* Same Data Type: All elements must be of same type.
* Index-based: Index starts from 0 to size - 1.
* Stored in contiguous memory.
* Faster access to elements using indices.

<h3 align="center"><b> 🔹 1D Array </b></h3>

<h5><b>✅ Declaration + Creation </b></h5>

```
int[] arr = new int[5];
```
<h5><b> ✅ Declaration + Initialization </b></h5>

``` 
int[] arr = {10, 20, 30};
```

<h5><b> 🔹 Length Property </b></h5>


```
arr.length
```



<h4 align="center"><b> 🔹 Traversing an Array </b></h4>

<h5><b> 🔸 Normal for loop </b></h5>

```
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

<h5><b> 🔸 Enhanced for-each loop </b></h5>

```
for (int value : arr) {
    System.out.println(value);
}
```

<h4 align="center"><b> 🔹 2D Arrays </b></h4>

<h5><b> ✅ Declaration + Creation </b></h5>

```
int[][] arr = new int[2][3];
```
2 rows, 3 columns.

<h5><b> ✅ Initialization </b></h5>

```
int[][] arr = {
    {1, 2, 3},
    {4, 5, 6}
};
```

<h5 align="center"><b>🔹 Traversing 2D Array (Row-Major Order) </b></h5>

```
for (int i = 0; i < arr.length; i++) {
    for (int j = 0; j < arr[i].length; j++) {
        System.out.print(arr[i][j] + " ");
    }
    System.out.println();
}
```
