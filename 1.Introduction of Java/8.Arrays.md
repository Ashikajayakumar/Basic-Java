<h3 align="center"><b> ðŸ”¹ What is an Array? </b></h3>

* An array is a collection of homogeneous (same type) elements stored in contiguous memory locations.
* It allows storing multiple values under a single variable name, accessed using index.`

<h4><b> ðŸ”¹ Key Features of Arrays: </b></h4>

* Fixed Size: Size must be known at compile time.
* Same Data Type: All elements must be of same type.
* Index-based: Index starts from 0 to size - 1.
* Stored in contiguous memory.
* Faster access to elements using indices.
