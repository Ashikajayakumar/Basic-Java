 
                                               Literal


* A literal have a constant value  fixed data type and often known as constant in Java.
* Literal assigned to a variable to provide a value to variable.


<h2 align="center"><b>A. Integer Literals</b></h2>

* Integer literals are numbers without decimals. They can be written in 3 formats:

<h5 align="center"> <b>1.Decimal (base 10) – normal numbers</b></h5>

```
int dec = 100;  // decimal literal
```

<h5 align="center"> <b>2.Octal (base 8) – starts with` 0 `</b></h5>

```
int oct = 010;  // octal literal → 010 = 8 in decimal
```

<h5 align="center"> <b>3.Hexadecimal (base 16) – starts with ` 0x or 0X`</b></h5>

```
int hex = 0x1A; // hexadecimal literal → 1A = 26 in decimal
```

Note: <b> Binary literals (base 2)</b> are also supported in Java 7+ using `0b or 0B`:

```
int bin = 0b1010; // binary literal → 10 in decimal
```

<h2 align="center"><b>B. Floating-Point Literals</b></h2>
 

Used for<b> numbers with a fractional part</b> (decimal numbers).

```
double price = 99.99;      // default is double
float weight = 65.5f;      // float literal must use 'f' or 'F'
```
* Default floating-point is double.
* Use <b>f/F</b> for float literals.
* Use <b>d/D</b> optionally for double literals.

<h2 align="center"><b> C. Character Literals </b></h2>

* <b>A single character enclosed in single quotes `' '`.</b>
```
char grade = 'A';
char symbol = '@';
char digit = '5';
```

* Can also use <b>escape sequences</b>:
```
char newline = '\n';
char tab = '\t';
char backslash = '\\';
```
<h2 align="center"><b> D. String Literals </b></h2>

* <b>A sequence of characters enclosed in double quotes `" "`.</b>
```
String name = "Ashika";
String greeting = "Hello, World!";
String empty = ""; // empty string literal
```


* Strings are objects in Java `(java.lang.String)`.

<h2 align="center"><b> E. Boolean Literals </b></h2>

Boolean literals represent <b>true/false </b> values.
```
boolean isJavaFun = true;
boolean isRaining = false;
```

Only `true` or `false` are valid literals for boolean type.

                             Type Casting

* Type casting is the process of <b>converting a variable from one data type to another</b>.
* In Java, there are two types of type casting:

           * Implicit Type Casting (Widening)
           * Explicit Type Casting (Narrowing)
  
<h2 align="center"><b>Implicit Type Casting (Widening)</b></h2>

* Definition: Automatic <b>conversion of a smaller data type to a larger data type</b>.
* Done <b>automatically</b> by Java.
* No data loss occurs.

Widening hierarchy (smaller → larger):
```
byte → short → int → long → float → double
char → int → long → float → double

```

Example:
```
public class WideningExample {
    public static void main(String[] args) {
        int myInt = 100;
        double myDouble = myInt;  // implicit casting (int → double)

        System.out.println("Integer value: " + myInt);
        System.out.println("Double value: " + myDouble);
    }
}
```

Output:
```
Integer value: 100
Double value: 100.0
```

<h2 align="center"><b> Explicit Type Casting (Narrowing)</b></h2>

* Definition: <b>Manual conversion</b> of a larger data type to a smaller data type.
* Must be done explicitly by the programmer using parentheses.
* Possible loss of data or precision.

Example:
```
public class NarrowingExample {
    public static void main(String[] args) {
        double myDouble = 9.78;
        int myInt = (int) myDouble;  // explicit casting (double → int)

        System.out.println("Double value: " + myDouble);
        System.out.println("Integer value: " + myInt);
    }
}
```

Output:
```
Double value: 9.78
Integer value: 9
```
