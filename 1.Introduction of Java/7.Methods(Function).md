<h2 align="center"><b>ğŸ”¹ Pass-by-Value in Java </b></h2>
<h4><b>ğŸ‘‰ Core Rule </b></h4>

<b>Java always passes arguments by value.</b>

This means:

* A <b>copy of the value</b> is sent to the method.
* Original variables are <b> not directly changed </b>.
