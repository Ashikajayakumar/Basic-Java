<h2 align="center"><b> What is Methods ?</b></h2>

* Java Methods are blocks of code that perform a specific task. 
* A method allows us to reuse code, improving both efficiency and organization.

<h4 align="center"><b>ðŸ”¹ Method Syntax</b></h4>


<h5><b> Example </b></h5>

```

    return a + b;
}
```

* ```returnType``` â†’ type of value returned
